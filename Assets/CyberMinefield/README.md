# Cyber Minefield

Unity implementation area for the Cyber Minefield project.

This folder turns the proposal in `Proyek GKV_K2_Kelompok 9.pdf` into Unity-facing planning artifacts and an implementation scaffold. Keep gameplay code under `Scripts`, game scenes under `Scenes`, reusable objects under `Prefabs`, and visual styling assets under `Materials`.

Core direction:
- Minesweeper-inspired grid puzzle.
- Cyber-security theme: malware, infected nodes, firewall/defuser markers, Nexus Core objective.
- Grid-based movement or selector movement, not free movement.
- MVP favors clear puzzle logic and readable tile numbers over heavy 3D production.
